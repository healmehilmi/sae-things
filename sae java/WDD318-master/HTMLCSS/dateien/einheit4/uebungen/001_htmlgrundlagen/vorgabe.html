Skriptum Was ist CSS? Wozu CSS? Initialwerte CSS Theorie Was ist CSS? CSS (Cascading Style Sheets) ist eine Stylesheet-Sprache für strukturierte Dokumente, mit der man das Aussehen von HTML Elementen festlegen bzw. beeinflussen kann. Es ist möglich Stylesheets für verschiedene Ausgabemedien (Bildschirm, Papier, Projektion, Sprache,..) zu schreiben. Wozu CSS? CSS dient zur Gestaltung der äußeren Form einer Website, das heißt man versieht ein einfaches HTML-Dokument mit Design. Dadurch soll sowohl die Funktionalität, als auch die Ästhetik derart aufgwertet werden, dass dem Benutzer ein besseres Erlebnis geboten wird. Initialwerte Jedes Element hat für jede denkbare Eigenschaft einen Standardwert (Initialwert). Diese Initialwerte werden im Browser vorgegeben und sind in den verschiedenen Browsern nahezu ident. Wäre dies nicht der Fall, müsste man jedes Mal beim Erstellen eines Stylesheets, für jedes Elemente alle Eigenschaften setzen. Mit Deklarationen ändert man also nur bestehende Eigenschaften und setzt nichts grundlegend Neues.
